?? Pizza Sales Analysis Dashboard (Power BI)
?? Project Overview
This project focuses on analyzing pizza sales data using Microsoft Power BI to gain meaningful business insights.
The dashboard helps understand sales performance, customer demand, popular pizzas, revenue trends, and order behavior through interactive visualizations.
The goal of this project is to demonstrate data cleaning, data modeling, DAX usage, and dashboard design skills using real-world sales data.

?? Objectives
* Analyze overall sales and revenue performance
* Identify top-selling and least-selling pizzas
* Track daily, monthly, and category-wise sales trends
* Understand customer ordering patterns
* Support business decision-making using visual insights

??? Tools & Technologies
* Power BI Desktop
* DAX (Data Analysis Expressions)
* Excel / CSV (Data Source)

?? Key Features & Insights
* ?? Total Revenue, Total Orders, Total Quantity Sold
* ?? Top 5 Pizzas by Sales & Quantity
* ?? Daily & Monthly Sales Trends
* ?? Category-wise and Size-wise Sales Analysis
* ? Peak Order Hours
* ?? Interactive filters for date, category, and pizza type

?? Data Processing Steps
1. Imported raw sales data into Power BI
2. Cleaned and validated data (handled duplicates, missing values)
3. Created relationships between tables
4. Built calculated columns and measures using DAX
5. Designed an interactive and user-friendly dashboard

?? How to Use
1. Download the repository
2. Open pizza sales.pbix using Power BI Desktop
3. Refresh data if needed
4. Explore insights using slicers and visuals

?? Use Cases
* Sales performance monitoring
* Product demand analysis
* Business reporting
* Learning Power BI dashboard development

?? Learning Outcomes
* Hands-on experience with Power BI
* Improved understanding of sales analytics
* Practical exposure to DAX and data modeling
* Experience working with real-world business data
